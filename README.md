# VM Rental - Car Rental Website

Welcome to **VM Rental**, a modern and comprehensive car rental web application designed to provide a seamless vehicle booking experience.

## 🚀 Features

- **Standardized UI:** A consistent, modern interface across all pages using Bootstrap.
- **Extensive Car Fleet:** Dedicated pages for various car categories, from luxury sedans to rugged SUVs.
- **Detailed Car Information:** Each car has a detail page showcasing its specifications, features, and pricing.
- **Easy Booking System:** A straightforward booking process that allows users to reserve vehicles quickly.
- **Modern Authentication:** Beautifully designed login and signup pages with dynamic video backgrounds.
- **Utility Tools:** Includes an EV Charging locator and a Fuel Estimator to help users plan their journeys.
- **Responsive Design:** Fully responsive layouts that work perfectly on desktops, tablets, and smartphones.

## 🛠️ Built With

- **HTML5 & CSS3:** For a robust and stylish structure.
- **Bootstrap 4.4.1:** For a responsive grid and pre-built components.
- **JavaScript & jQuery:** For interactive elements and dynamic functionality.
- **Font Awesome 5:** For a rich set of icons.
- **Google Fonts:** Using the 'Oswald' and 'Rubik' typefaces for a clean, professional look.
- **Sass (SCSS):** For organized and maintainable styling.

## 📂 Project Structure

- `/html`: Contains all the website pages, including car detail and booking pages.
- `/css`: Includes both compiled CSS and Bootstrap stylesheets.
- `/image`: A collection of high-quality car images, icons, and background videos.
- `/js`: Custom JavaScript files for handling client-side logic.
- `/lib`: Third-party libraries like Owl Carousel and Tempus Dominus for advanced UI features.

## 🚗 How to Use

1.  **Browse Cars:** Visit the `Categories.html` page to see the available car categories.
2.  **View Details:** Click on any car to see its detailed specifications on its dedicated landing page.
3.  **Book a Ride:** From the car landing page, click "Reserve Now" to go to the booking form.
4.  **Manage Account:** Use the `Login.html` and `Signup.html` pages to access your personal account.
